{{- /* Browser-language auto-detect for the root URL only. Static-site approach: we can't choose a locale server-side, so the root /index.html redirects to /en/ by default (the above and the location.replace in the noscript fallback). Before that fires, this inline script checks navigator.languages. If the visitor's preferred language matches one of our 9 supported locales and ISN'T English, we replace the redirect target. The check is bounded: - Bots: skipped via a simple UA check so search engines see /en/ - Repeat visits: localStorage 'auto-lang-applied' set after the first run, so the redirect doesn't keep firing (e.g. if user manually navigates back to root) - User picked a language: the lang-switcher in static/js/app.js stores 'site-lang' in localStorage; if set, we honour it over navigator.languages */ -}}
